# Team introduction application #
 ## Functions ##
 * Add user
 * Delete user
 * Update user information such as image, firstname, lastname and so on.
 
  ## Installation ##
 To run this application you will need to install CocoaPods. Go to the project directory, run command: 
 
 `pod install`
